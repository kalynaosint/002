📖 使用说明
首次使用流程：
1. 创建 GitHub 仓库并启用 GitHub Pages（Settings → Pages → Deploy from branch: main）
2. 将所有文件（index.html、admin.html、data/ 目录）推送到仓库
3. 在上方配置 GitHub 信息，填入 Personal Access Token
4. 点击「测试连接 & 加载数据」
5. 开始上传图片和编写报告

每日工作流程：
1. 截图或保存前线图 → 粘贴到「上传图片」标签页
2. 选择方向和日期，点击上传
3. 重复上传各个方向
4. 切换到「编辑报告」，输入当日文字综述并保存
